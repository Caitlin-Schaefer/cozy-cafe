OOP Project:
Project Name: Cozy Cafe

Project Overview and Use: 

Cozy Cafe is a simplified business simulator, allowing individuals of all ages to learn and develop
their understanding of the world of business in a low-stakes environment. The program includes
features of customer service (taking and handling orders), managing inventory, and growing the
cafe menu. In this, the owner (user) can complete the customer’s order through different
challenges, enabling the expansion of the cafe over time. This project aims to be fun and engaging
while allowing the user to learn about money management and business with ease.
