# Object Oriented Cozy Cafe

Cozy Cafe is a simplified business simulator created in C++. It allows individuals of all ages to learn and develop
their understanding of the world of business in a low-stakes environment. 

## **Features**
- Save Slots (1, 2, or 3 slots)
- Progressively increasing difficulty
- Easy to extend design for adding new features or challenges
- Customer Service Management:
  - Take and Handle Customer Orders through Interactive Challenges
  - Track Order Accuracy and Speed to simulate real-world service.
- Inventory Management:
  - Monitor stock levels for ingredients and supplies
  - Restock strategically to meet customer demand and maximise profit.
- Menu Expansion:
  - Unlock new Items and Recipes as the Cafe grows.
- Business and Money Management:
  - Simulate Budgeting, expenses and profits.
  - Make strategic decisions to expand the cafe and grow your business. 

## Requirements
- C++11 Compiler

## Compilation

To compile the project, follow these steps:

2. Clone the repository:
   ```
   git clone https://github.com/Caitlin-Schaefer/cozy-cafe.git
   ```
3. Navigate to the project directory:
   ```
   cd cozy-cafe/CPP\ Files
   ```
4. Run the following command to compile the project:
   ```
   make
   ```
